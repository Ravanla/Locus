## Changes

### Added

- Added a Git history search window in Collab for searching commits and stashes by file/path query, regex, author, and date range.
- Added live progress events for `unity_execute` and `unity_run_states` in chat tool call panels.
- Added a display setting to hide Git terminal command suggestions.
- Added knowledge inspection tools to the Git Agent.

### Fixed

- Fixed Git terminal pending input by rendering ask-user and tool confirmation prompts inline.
- Fixed saved reasoning effort handling so session and agent context changes keep the user's persisted default.
- Fixed `unity_yaml_read` enum serialization when Unity does not provide enum display names.
- Fixed Knowledge lexical rebuild progress for small single-document updates and large batch progress totals.
- Fixed Knowledge selection cleanup after deleting documents or folders.
- Filtered third-party TRACE logs while debug mode is enabled.
- Improved `unity_yaml_read` tool header summaries with full file and object paths.

### Removed

- Removed the Claude subscription OAuth sign-in entry point while keeping logout available for existing accounts.
