## Changes

### Added

- Added startup tracing across the Rust backend and Vue frontend, with the main window shown after the first mounted frame.
- Added deferred startup work for tab preloading, recent workspace loading, Unity monitoring, asset reconciliation, and knowledge indexing.
- Added inline model setup in onboarding for custom endpoints and OpenAI Codex device auth.
- Added connection testing for custom endpoints during onboarding.
- Added visible vector runtime startup state in Knowledge, including a compact loading indicator while local embeddings start.

### Fixed

- Cached Git and Python runtime discovery so Settings can reuse discovered runtimes without repeating blocking scans.
- Moved Git runtime discovery off the blocking command path.
- Added onboarding project-opening feedback and disabled navigation while the selected project is being opened.
- Kept Knowledge list and overview reads responsive while the embedding manager is busy.

### Removed

- No changes.
