## 变更列表

### 新增

- 新增 Rust 后端与 Vue 前端启动阶段追踪，主窗口会在首帧挂载后显示。
- 新增延后启动任务，用于标签预加载、最近工作区加载、Unity 监听、资产索引校准与知识索引。
- Onboarding 支持内联配置自定义模型端点与 OpenAI Codex 设备授权。
- Onboarding 支持测试自定义端点连接。
- Knowledge 显示向量运行时启动状态，本地 embedding 启动时会展示轻量加载提示。

### 修复

- Git 与 Python 运行时发现结果会缓存，Settings 可复用发现结果，减少重复阻塞扫描。
- Git 运行时发现移到后台阻塞任务中执行。
- Onboarding 打开项目时会显示处理中状态，并在切换完成前禁用导航。
- Knowledge 列表与概览读取在 embedding manager 繁忙时保持响应。
