## 变更列表

### 新增

- Collab 新增 Git 历史搜索窗口，支持按文件/路径、正则、作者和日期范围搜索 commit 与 stash。
- `unity_execute` 和 `unity_run_states` 的工具调用面板新增实时进度事件。
- 设置中新增隐藏 Git 终端命令候选项的显示开关。
- Git Agent 新增通过知识工具检查 Locus 知识文档的能力。

### 修复

- Git 终端现在以内联形式展示用户提问和工具确认。
- 修复推理强度保存逻辑，Session 与 Agent 的上下文强度不会覆盖用户默认选择。
- 修复 Unity 缺少 enum display name 时 `unity_yaml_read` 的序列化。
- 修复 Knowledge lexical rebuild 的小更新静默处理和大批量进度总量。
- 修复删除知识文档或目录后的选中状态清理。
- Debug 模式下过滤第三方 TRACE 日志。
- 改进 `unity_yaml_read` 工具标题摘要，展示完整文件与对象路径。

### 移除

- 移除 Claude 订阅 OAuth 登录入口，已登录账号仍可在设置中退出。
